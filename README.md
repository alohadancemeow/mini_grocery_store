# Project Mobile Application.

### Mini Grocery Store 🍉:
A small grocery store that serves fresh food for customers who want good health. 💪
- using flutter + firebase

🎏 Contain 4 pages:
- Intro page
- Home page
- Item detail page
- Cart page

📚 Features:
- Add item to cart
- Remove item from cart
- Check out
